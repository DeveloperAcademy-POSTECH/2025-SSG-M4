
# Swift - for 루프 (For loops – Swift in Sixty Seconds 요약)

### 개요

*   Swift에서 **루프는 코드를 반복해서 실행하는 메커니즘**입니다 [1].
*   **조건이 false가 될 때까지 코드를 반복적으로 실행**합니다 [1].
*   Swift에서 가장 흔하게 사용되는 루프는 **`for` 루프**입니다 [1].

### for 루프의 작동 방식

*   `for` 루프는 **배열이나 범위(ranges)를 순회**할 때 사용됩니다 [1].
*   루프가 **한 번 돌 때마다 순회 대상(배열 또는 범위)에서 하나의 항목을 꺼내와 상수에 할당**합니다 [1].

#### 예시 1: 범위(Range) 순회

*   **숫자 범위**를 순회할 수 있습니다 [1].
*   예시: `let count = 1 through 10` (1부터 10까지의 범위) [1]
*   이 범위를 `for` 루프로 순회하며 각 항목을 출력하는 방법:
    ```swift
    for a number in count {
        print "number is \(number)"
    }
    ```
    *   여기서 `a number`는 범위에서 꺼내온 현재 항목을 담는 상수입니다.

#### 예시 2: 배열(Array) 순회

*   **배열**도 `for` 루프로 순회할 수 있습니다 [1].
*   예시: `let albums = ["red", "1989", "reputation"]` (문자열 배열) [1]
*   이 배열을 `for` 루프로 순회하며 각 항목을 출력하는 방법:
    ```swift
    for album in albums {
        print "\(album) is on Apple music"
    }
    ```
    *   여기서 `album`은 배열에서 꺼내온 현재 항목을 담는 상수입니다.

### 상수 사용이 필요 없을 때 (Underscore `_` 사용)

# Swift - while 루프 (While loops)

### 개요

*   Swift에서 루프(loops)는 코드를 **반복해서 실행하는 메커니즘**입니다.
*   `for` 루프 외에 Swift에서 루프를 작성하는 **또 다른 방법**은 `while` 루프를 사용하는 것입니다.

### while 루프의 작동 방식

*   `while` 루프는 **확인할 조건**을 지정합니다.
*   제시된 **조건이 참(true)인 동안** `while` 루프 내부의 코드를 **반복해서 실행**합니다.
*   조건이 **거짓(false)이 되는 순간** 루프의 실행이 **종료**됩니다.

### 예시: 숨바꼭질 숫자 세기

*   `while` 루프를 사용하여 숨바꼭질 게임에서 아이가 숫자를 세는 상황을 시뮬레이션할 수 있습니다.
*   **1부터 시작하여 20까지 (20 포함)** 각 숫자를 출력하고, 20까지 센 후 루프를 종료하고 특정 메시지를 출력하는 예시입니다.

#### 코드

```swift
var number = 1 // 숫자를 저장할 변수를 선언하고 1로 초기화

while number <= 20 { // number가 20보다 작거나 같은 동안 루프를 계속 실행
    print(number) // 현재 number 값을 출력
    number += 1 // number 값을 1 증가 (다음 반복을 위해)
} // number가 21이 되면 조건(number <= 20)이 거짓이 되어 루프 종료

print("Ready or not, here I come!") // 루프 종료 후 이 문장을 출력
````

설명

1.var number = 1: number라는 변수를 선언하고 초기값을 1로 설정합니다. while 루프는 반복 횟수를 세거나 조건을 변경하기 위해 변수를 사용하는 경우가 많습니다 (for 루프는 주로 컬렉션의 각 항목을 상수에 할당하며 사용).

2.while number <= 20: 이것이 while 루프의 조건입니다. number의 값이 20보다 작거나 같을 때만 루프 안의 코드가 실행됩니다.

3.print(number): 루프가 한 번 돌 때마다 현재 number 값을 출력합니다.

4.number += 1: number의 값을 1 증가시킵니다. 이 과정이 없으면 number 값은 계속 1에 머물러 조건 (number <= 20)이 항상 참이 되므로 무한 루프에 빠지게 됩니다. while 루프에서는 루프 조건에 영향을 주는 변화가 루프 본문 안에 반드시 포함되어야 합니다.

5.print("Ready or not, here I come!"): number가 21이 되어 while number <= 20 조건이 거짓이 되면 루프가 종료되고, 이 문장이 실행됩니다.

while 루프는 for 루프처럼 정해진 컬렉션을 순회하는 것보다는, 특정 조건이 충족될 때까지 또는 조건이 더 이상 참이 아닐 때까지 작업을 반복해야 할 때 유용하게 사용됩니다.

# 🔁 Swift의 `repeat-while` 루프 이해하기

Swift에서 루프(반복문)를 작성하는 세 번째 방법으로 `repeat-while` 루프가 있습니다. 이 루프는 다른 루프들만큼 자주 사용되지는 않지만, 배우기 매우 간단해서 알아두면 유용합니다. `repeat-while` 루프는 **조건을 루프의 마지막에서 확인한다는 점을 제외하면 `while` 루프와 거의 동일**합니다.

---
### 📝 `repeat-while` 루프의 기본 구조와 예시

`while` 루프는 조건을 먼저 검사하고 루프 본문을 실행할지 결정하는 반면, `repeat-while` 루프는 일단 **루프 본문을 최소 한 번 실행한 후, 마지막에 조건을 검사**하여 반복 여부를 결정합니다.

숨바꼭질 예제를 `repeat-while` 루프로 작성하면 다음과 같습니다:

Swift

```
var number = 1
repeat {
    print(number)
    number += 1
} while number <= 20
print("Ready or not, here I come!")
```

**코드 설명:**

1. `var number = 1`: 변수 `number`를 1로 초기화합니다.
2. `repeat { ... }`: `{ ... }` 안의 코드를 먼저 실행합니다.
    - `print(number)`: 현재 `number` 값을 출력합니다.
    - `number += 1`: `number` 값을 1 증가시킵니다.
3. `while number <= 20`: 루프 본문 실행 후, `number`가 20보다 작거나 같은지 조건을 검사합니다.
    - 조건이 `true`이면 `repeat` 블록으로 돌아가 다시 실행합니다.
    - 조건이 `false`이면 루프를 종료합니다.
4. `print("Ready or not, here I come!")`: 루프가 종료된 후 실행됩니다.

---

### 🤔 `while` 루프와의 핵심 차이: 최소 실행 횟수 보장

`repeat-while` 루프와 `while` 루프의 가장 중요한 차이점은 **루프 내부 코드가 최소 한 번은 반드시 실행된다는 것**입니다. `repeat-while` 루프는 조건을 루프의 _끝에서_ 확인하기 때문에, 루프 안의 코드는 조건과 관계없이 일단 한 번은 실행됩니다. 반면, `while` 루프는 조건을 루프의 _처음에서_ 확인하므로, 조건이 처음부터 거짓이면 루프 내부 코드가 한 번도 실행되지 않을 수 있습니다.

#### 예시를 통한 비교

**1. `while` 루프의 경우 (조건 선행 검사)**

다음 `print()` 함수는 `while` 조건이 `false`이므로 **절대 실행되지 않습니다.**

Swift

```
while false {
    print("This is false") // 이 코드는 실행되지 않습니다.
}
```

> Xcode는 심지어 위 `print()` 줄이 절대 실행되지 않을 것이라고 경고해 줄 것입니다. 이는 `while` 루프가 코드를 실행하기 _전에_ 조건을 먼저 검사하기 때문입니다.

**2. `repeat-while` 루프의 경우 (조건 후행 검사)**

반면에, 다음 `print()` 함수는 **한 번 실행됩니다.** `repeat` 블록 안의 코드가 먼저 실행된 후, `while` 조건을 검사하기 때문입니다.

Swift

```
repeat {
    print("This is false") // 이 코드는 한 번 실행됩니다.
} while false
```

> 이 경우, `print("This is false")`가 먼저 실행되고, 그 후에 `while false` 조건이 검사됩니다. 조건이 `false`이므로 루프는 더 이상 반복하지 않고 종료됩니다.

---

### 📌 `repeat-while` 루프 요약 및 활용

- `repeat-while` 루프는 루프 본문을 **최소 한 번 실행**하고 싶을 때 유용합니다.
- 조건 검사는 루프 본문이 실행된 **후에** 이루어집니다.
- `while` 루프와 유사하지만, 조건 검사 시점이 다릅니다.

이처럼 `repeat-while` 루프는 "일단 실행하고, 계속할지 나중에 결정하자"는 방식의 반복 처리에 적합합니다. 예를 들어, 사용자로부터 입력을 받아 유효성을 검사할 때, 최소 한 번은 입력을 받아야 하는 경우 등에 활용될 수 있습니다.2

## Swift - 루프 탈출하기 (Exiting loops – `break` 키워드)

### 개요

*   Swift에서는 루프(loops) 실행 중에 **특정 조건이 충족되었을 때** 남은 반복 과정을 모두 건너뛰고 **즉시 루프를 종료**할 수 있는 방법이 있습니다.
*   이를 위해 사용되는 키워드가 바로 **`break`** 입니다 [1].

### `break` 키워드의 작동 방식

*   `break` 키워드를 만나면 현재 실행 중인 루프는 **즉시 중단**됩니다 [1].
*   루프 본문 내에 `break`가 포함된 문장 이후의 코드는 해당 반복 주기에서 실행되지 않으며, 루프 자체도 더 이상 다음 반복으로 진행되지 않고 **종료**됩니다 [1].
*   프로그램 실행은 루프가 종료된 **바로 다음 코드**부터 계속됩니다.

### 예시: 로켓 발사 카운트다운 중 루프 탈출하기

*   `while` 루프를 사용하여 로켓 발사 카운트다운을 시뮬레이션하는 상황을 통해 `break`의 사용법을 이해할 수 있습니다.

#### 1. `break`가 없는 일반적인 `while` 루프 (카운트다운 예시)

*   이 루프는 10부터 시작하여 0까지 숫자를 센 후 "Blast off!"를 출력하며 종료됩니다.

```swift
var countDown = 10 // 카운트다운 시작 숫자를 변수에 저장

while countDown >= 0 { // countDown이 0보다 크거나 같은 동안 루프 실행
    print(countDown) // 현재 countDown 값을 출력
    countDown -= 1 // countDown 값을 1 감소
} // countDown이 -1이 되면 조건(countDown >= 0)이 거짓이 되어 루프 종료

print("Blast off!") // 루프 종료 후 실행
````

•설명: 이 코드는 countDown 변수가 10에서 시작하여 0까지 1씩 감소하며 각 숫자를 출력합니다. countDown이 -1이 되는 순간 while 조건이 false가 되어 루프가 정상적으로 종료되고, 마지막 줄의 print("Blast off!")가 실행됩니다.

2. 를 사용하여 루프 중간에 탈출하기 (카운트다운 조기 종료 예시)

•사령관인 우주 비행사가 카운트다운 도중 지루해져서 남은 카운트다운을 건너뛰고 바로 발사하기로 결정하는 상황을 가정합니다1.

•이 경우, 카운트다운 숫자가 특정 값(예: 4)에 도달했을 때 루프를 즉시 중단하도록 break 키워드를 사용합니다1.

```
// Note: This example assumes countDown was reset or is a new variable.
// If using the variable from the previous example, its value would be -1,
// and the loop condition `countDown >= 0` would immediately be false.
// Let's assume countDown is reset to 10 for this example.
var countDown = 10 // 예시를 위해 countDown을 다시 10으로 설정

while countDown >= 0 { // countDown이 0보다 크거나 같은 동안 루프 실행 시도
    print(countDown) // 현재 countDown 값을 출력

    // 카운트다운이 4에 도달했는지 확인하는 조건 추가 [1]
    if countDown == 4 {
        print("I'm bored. Let's go now!") // 특정 메시지 출력 [1]
        break // **여기서 루프를 즉시 중단합니다! [1]**
    }

    // break 때문에 이 부분은 countDown이 4일 때는 실행되지 않습니다.
    countDown -= 1
}

// 루프가 break에 의해 종료되었으므로,
// 위 예시의 마지막 print("Blast off!") 문장은 실행되지 않습니다.
// (제공된 코드 예시에는 이 줄이 없으며, 설명에 따라 루프가 스킵되었음을 알 수 있습니다)
```

•

설명:

1.루프는 countDown이 10, 9, 8, 7, 6, 5일 때 정상적으로 숫자를 출력하고 countDown 값을 1씩 감소시킵니다.

2.countDown이 4가 되면 if countDown == 4 조건이 **참(true)**이 됩니다1.

3.조건문 안의 코드가 실행됩니다: "I'm bored. Let's go now!" 메시지가 출력됩니다1.

4.이어서 break 키워드가 실행됩니다1.

5.break를 만나는 순간, while 루프는 즉시 중단됩니다1.

6.현재 반복 주기에서 break 뒤에 있는 countDown -= 1 코드는 실행되지 않습니다.

7.루프 자체가 종료되었으므로, countDown은 4에서 멈추고 더 이상 감소하지 않으며, 루프 조건(while countDown >= 0) 검사도 중단됩니다. 루프 이후의 코드가 있다면 그 코드부터 실행됩니다.

•결과: 이 코드는 10, 9, 8, 7, 6, 5, 4를 출력한 후 "I'm bored. Let's go now!" 메시지를 출력하고 종료됩니다1. 3, 2, 1, 0 및 "Blast off!"는 출력되지 않습니다1.

핵심 정리

•break 키워드는 루프의 실행을 즉시 중단시키는 역할을 합니다1.

•주로 if 문과 함께 사용하여 특정 조건이 충족되었을 때 루프를 빠져나오는 데 사용됩니다.

•break가 실행되면 해당 루프 본문의 남은 부분은 건너뛰어지고, 루프 자체도 종료됩니다1.

# Swift - 중첩 루프 탈출하기 (Exiting Nested Loops)

## 🎯 개요 (Overview)

* **중첩 루프(Nested loop)**란 하나의 루프 안에 또 다른 루프가 포함된 구조를 의미합니다.
* 중첩 루프 사용 시, 가장 안쪽 루프뿐 아니라 바깥쪽 루프까지 여러 루프를 동시에 탈출해야 하는 상황이 발생할 수 있습니다.

---

## ⚠️ 일반적인 `break`의 한계 (Limitations of a Normal `break`)

* 일반적인 `break` 키워드는 해당 `break` 문이 포함된 **가장 가까운 루프 하나만 종료**시킵니다.
* 따라서 중첩 루프의 안쪽 루프에서 `break`를 사용하면, 안쪽 루프는 종료되지만 **바깥쪽 루프는 계속 실행**됩니다. 바깥쪽 루프는 다음 반복으로 넘어가 안쪽 루프를 다시 시작하게 됩니다.

---

## ✨ 여러 루프를 동시에 탈출하는 방법: 레이블(Labels) 사용 (Exiting Multiple Loops: Using Labels)

* Swift에서는 루프에 **레이블(Label)**을 지정하고, `break` 키워드와 함께 이 레이블을 사용하여 **지정한 레이블의 루프와 그 안의 모든 중첩 루프를 한 번에 종료**할 수 있습니다.

---

## 🛠️ 사용 단계 (Steps to Use)

1.  **바깥쪽 루프에 레이블 지정**: 탈출하고 싶은 가장 바깥 루프 앞에 **레이블 이름**을 쓰고 **콜론(:)**을 붙입니다.
    * 예시: `outerLoop: for ...`
2.  **안쪽 루프 내 조건 확인**: 루프를 탈출하고 싶은 특정 조건(예: `if` 문)을 안쪽 루프 안에 작성합니다.
3.  **`break`와 레이블 사용**: 조건이 참(true)일 때 실행될 코드 블록 안에 `break` 키워드 뒤에 지정한 **레이블 이름**을 붙여 사용합니다.
    * 예시: `break outerLoop`

---

## 🔢 예시: 구구단 계산 중 특정 결과 찾기 (Example: Finding a Specific Result in Multiplication Tables)

1부터 10까지의 숫자에 대한 구구단을 계산하는 중첩 루프에서, 계산 결과(곱셈 값)가 **50이 되는 순간** 전체 루프를 종료하는 예시입니다.

### 1. `break`를 사용하지 않은 기본적인 중첩 루프 (Basic Nested Loop - No `break`)

이 코드는 1단부터 10단까지 모든 곱셈 결과를 출력합니다.

```swift
// 1단부터 10단까지의 구구단 출력
for i in 1...10 { // 바깥쪽 루프: 곱하는 첫 번째 숫자 (1부터 10까지)
    for j in 1...10 { // 안쪽 루프: 곱하는 두 번째 숫자 (1부터 10까지)
        let product = i * j // 두 숫자의 곱셈 결과 계산
        print ("\(i) * \(j) is \(product)") // 결과를 출력
    } // 안쪽 루프 종료 후 바깥쪽 루프의 다음 i 값으로 넘어감
}
````

- **설명**: 바깥쪽 `for` 루프가 `i` 값을 1부터 10까지 변경합니다. 바깥쪽 루프가 한 번 반복될 때마다 안쪽 `for` 루프가 `j` 값을 1부터 10까지 반복하며 곱셈을 수행하고 결과를 출력합니다. 안쪽 루프가 10번 반복 완료된 후에야 바깥쪽 루프가 다음 `i` 값으로 넘어갑니다.

### 2. 레이블과 `break`를 사용하여 중첩 루프 중간에 탈출하기 (Exiting with Label and `break`)

구구단 계산 중 결과가 50이 되는 순간, 나머지 계산 과정을 모두 건너뛰고 즉시 전체 중첩 루프를 종료합니다. ("황소의 눈(bullseye)"에 맞춘 것에 비유)

```Swift
// 바깥쪽 루프에 'outerLoop' 레이블 지정
outerLoop: for i in 1...10 { // 바깥쪽 루프 (레이블 지정)
    for j in 1...10 { // 안쪽 루프
        let product = i * j // 곱셈 결과 계산
        print ("\(i) * \(j) is \(product)") // 현재 결과 출력

        // 곱셈 결과가 50인지 확인하는 조건문 추가
        if product == 50 {
            print("It's a bullseye!") // 50을 찾았다는 메시지 출력
            break outerLoop // 'outerLoop' 레이블이 지정된 루프와 그 안의 모든 루프를 즉시 중단!
        }
    } // 안쪽 루프의 끝 (break outerLoop가 실행되면 여기까지 오지 않고 전체 루프 종료)
} // 바깥쪽 루프의 끝 (break outerLoop가 실행되면 여기까지 오지 않고 전체 루프 종료)
```

- **설명**:
    1. 바깥쪽 `for` 루프에 `outerLoop:` 라는 레이블을 붙였습니다.
    2. 안쪽 `for` 루프에서 `product` 값이 계산될 때마다 `if product == 50` 조건문을 확인합니다.
    3. `i`가 5, `j`가 10이 되어 `product`가 50이 되는 순간, 조건문이 **참(true)**이 됩니다.
    4. 조건문 안의 코드 `print("It's a bullseye!")` 와 `break outerLoop` 가 실행됩니다.
    5. **`break outerLoop`**가 실행되는 즉시, `outerLoop` 레이블이 붙은 바깥쪽 루프와 현재 실행 중이던 안쪽 루프 모두 즉시 중단됩니다.
    6. 결과적으로 `5 * 10 = 50`이 출력된 후 메시지가 출력되고 루프가 종료됩니다. `5 * 10` 이후의 5단 계산 및 6단부터 10단까지의 모든 계산은 건너뛰어집니다.

---

## 🔑 핵심 정리 (Key Summary)

- 중첩 루프에서 가장 가까운 루프 하나만 탈출하려면 **`break`**를 사용합니다.
- 중첩 루프에서 특정 레이블이 지정된 루프와 그 안의 모든 루프를 한 번에 탈출하려면 해당 레이블 이름과 함께 **`break labelName`**을 사용합니다.
- 레이블은 주로 바깥쪽 루프에 지정하여 안쪽 루프의 특정 조건에서 전체 중첩 루프를 종료할 때 유용하게 사용됩니다.

# Swift - 루프에서 특정 항목 건너뛰기: `continue` 사용하기 (Skipping Items in a Loop: Using `continue`)

## 🎯 개요 (Overview)

* 이전에 보셨듯이, **`break`** 키워드는 루프 자체를 완전히 종료시킵니다.
* 하지만 **현재 루프의 특정 항목(iteration)**만 건너뛰고 다음 항목으로 **계속 진행**하고 싶을 때가 있습니다.
* 이럴 때 사용하는 키워드가 바로 **`continue`** 입니다.

---
## 🔄 `continue`와 `break`의 차이 (Difference between `continue` and `break`)

* **`break`**: `break`가 포함된 **루프 전체를 즉시 중단**하고 루프 다음 코드로 넘어갑니다. (앞서 학습한 레이블과 함께 사용하면 중첩 루프 여러 개를 한 번에 탈출할 수도 있습니다.)
* **`continue`**: `continue`가 실행되면 현재 루프의 **나머지 코드 실행을 건너뛰고** 루프의 다음 반복(next iteration)으로 즉시 넘어갑니다. 루프 자체는 계속 실행됩니다.

---
##  przykład `continue` 사용 예시: 홀수 건너뛰고 짝수만 출력하기 (Example: Skipping Odd Numbers, Printing Only Even Numbers)

1부터 10까지 숫자를 반복하면서, **홀수는 건너뛰고 짝수만 출력**하는 예시를 통해 `continue`의 동작 방식을 이해할 수 있습니다.

### 사용 단계 및 설명

1.  **루프 생성**: 1부터 10까지 숫자를 반복하는 `for` 루프를 작성합니다.
    ```swift
    for i in 1...10 {
        // 루프 내용
    }
    ```
2.  **조건 확인**: 루프를 건너뛰고 싶은 조건(여기서는 숫자가 홀수인지 확인하는 조건)을 `if` 문으로 작성합니다. 숫자가 홀수인지 확인하기 위해 **나머지 연산자(`%`)**를 사용합니다.
    * **나머지 연산자(`%`) 설명**: 나머지 연산자는 왼쪽 수를 오른쪽 수로 나눈 후 남는 **나머지 값**을 계산합니다.
    * `i % 2`는 `i`를 2로 나눈 나머지를 구합니다.
    * 만약 `i`를 2로 나눈 나머지가 **1**이라면, 그 수는 **홀수**라는 뜻입니다.
3.  **`continue` 사용**: `if` 조건(`i % 2 == 1`, 즉 홀수이면)이 참(true)일 때, `continue` 키워드를 실행하여 현재 반복의 나머지 부분을 건너뛰도록 합니다.
4.  **건너뛰지 않을 경우 실행될 코드**: `if` 조건에 해당하지 않을 경우(즉, 숫자가 짝수일 경우), `continue`가 실행되지 않으므로 루프의 나머지 코드(여기서는 `print(i)`)가 정상적으로 실행됩니다.

### 전체 코드 예시

```swift
// 1부터 10까지 루프를 돌며 홀수는 건너뛰고 짝수만 출력
for i in 1...10 { // 1부터 10까지 숫자를 i에 순서대로 할당하며 루프 실행
    
    // i가 짝수인지 홀수인지 확인: 2로 나누었을 때 나머지가 1이면 홀수
    if i % 2 == 1 { // 만약 i가 홀수이면...
        // 현재 i 값에 대한 나머지 코드(print(i)) 실행을 건너뛰고
        // 즉시 다음 i 값(루프의 다음 반복)으로 이동합니다.
        continue 
    }
    
    // i가 홀수가 아니면 (즉, 짝수이면) 이 코드가 실행됩니다.
    print(i) 
}

// 이 코드를 실행하면 다음과 같은 결과가 출력됩니다:
// 2
// 4
// 6
// 8
// 10
````

- **설명**: 루프가 1부터 시작합니다.
    - `i`가 1일 때: `1 % 2 == 1` (참) -> `continue` 실행 -> `print(1)` 건너뛰고 다음 `i` 값(2)으로 이동.
    - `i`가 2일 때: `2 % 2 == 1` (거짓) -> `continue` 실행 안됨 -> `print(2)` 실행.
    - `i`가 3일 때: `3 % 2 == 1` (참) -> `continue` 실행 -> `print(3)` 건너뛰고 다음 `i` 값(4)으로 이동.
    - ...이 과정이 10까지 반복되어 결국 짝수만 출력됩니다.

---

## 🔑 핵심 정리 (Key Summary)

- **`continue`**는 루프의 현재 반복(current iteration)만 건너뛰고 다음 반복으로 넘어갈 때 사용합니다.
- **`break`**는 루프 전체를 종료할 때 사용합니다.
- `continue`는 특정 조건을 만족하는 항목을 처리하지 않고 그냥 넘어가고 싶을 때 유용하게 사용됩니다.


# Swift - 무한 루프 (`while true`)와 루프 종료 (`break`) (Infinite Loops and Loop Termination)

## 🎯 개요 (Overview)

* 루프에는 정해진 횟수만큼 반복하는 경우도 있지만, **특정 조건이 될 때까지 계속 반복**하거나 **앱이 실행되는 동안 끊임없이 실행**되어야 하는 경우도 있습니다.
* 이처럼 **끝이 없거나 특정 조건이 만족될 때만 종료되는 루프**를 **무한 루프 (Infinite loops)** 라고 합니다.
* Swift에서는 주로 `while` 루프를 사용하여 무한 루프를 만듭니다.
* 아이폰의 모든 앱들도 무한 루프를 사용합니다. 앱이 시작된 후 사용자가 종료할 때까지 이벤트를 지속적으로 감시하며 실행 상태를 유지하기 때문입니다.

---

## 🔄 `while true` 로 무한 루프 만들기 (Creating an Infinite Loop with `while true`)

* 무한 루프를 만드는 가장 간단한 방법은 `while` 문의 조건에 **`true`**를 사용하는 것입니다.
    ```swift
    while true {
        // 이 안에 있는 코드는 계속 반복됩니다.
    }
    ```
* `true`는 항상 참(True)인 값이기 때문에, `while true`의 조건은 **절대 거짓(False)이 되지 않습니다.** 따라서 이 루프는 **영원히 반복**됩니다.

---

## ⚠️ 중요 경고: 반드시 종료 조건이 필요합니다! (Important Warning: Termination Condition is Essential!)

* `while true`로 무한 루프를 만들 때 **가장 중요한 점**은 **루프를 빠져나올 수 있는 종료 조건을 반드시 포함**해야 한다는 것입니다.
* 만약 종료 조건이 없으면, 루프는 영원히 끝나지 않으며 프로그램이 멈추거나 응답하지 않게 될 수 있습니다.
* 루프를 종료하기 위해 **`break`** 키워드를 사용합니다. 특정 조건이 만족되었을 때 `break`를 실행하면 루프가 즉시 중단됩니다.

---

## 🎹 예시: 존 케이지의 '4분 33초' 음악 표현하기 (Example: Expressing John Cage's '4'33"')

무한 루프와 `break`의 사용을 이해하기 위해, 존 케이지(John Cage)의 유명한 음악 작품인 **'4분 33초 (4'33")'**를 Swift 코드로 표현하는 예시를 사용해 보겠습니다. 이 곡은 특이하게도 **4분 33초 동안 아무 소리도 내지 않고 연주**되는 것으로 유명합니다. 우리는 이 곡의 '침묵'을 **일정 횟수만큼 공백(` `)을 출력**하는 것으로 표현하고, 정해진 횟수만큼 반복한 후에 루프를 종료하도록 만들 수 있습니다.

### 예시 코드 및 설명

```swift
// '4분 33초' 곡을 표현하기 위한 카운터 변수 초기화
var counter = 0 // 루프가 몇 번 반복되었는지 셀 변수입니다. 처음에는 0으로 시작합니다.

// 무한 루프 시작: 조건이 항상 true이므로 계속 반복됩니다.
while true { 
    
    // 1. '침묵'의 한 순간을 표현하기 위해 공백을 출력합니다.
    print(" ") // 매 반복마다 공백 문자 하나를 출력합니다.
    
    // 2. 루프가 한 번 실행될 때마다 카운터를 1 증가시킵니다.
    counter += 1 // counter = counter + 1 과 같은 의미입니다.

    // 3. 루프 종료 조건 확인
    // 카운터가 273에 도달했는지 확인합니다.
    if counter == 273 { // 만약 카운터가 273과 같으면...
        // 4. 조건이 만족되면 루프를 즉시 종료합니다.
        break // 이 키워드를 만나면 while 루프 전체가 중단됩니다.
    }
    
    // break가 실행되지 않았다면, 루프는 다음 반복으로 넘어갑니다.
}

// break에 의해 루프가 종료되면, 이 코드 아래의 다음 줄부터 실행이 계속됩니다.
print("음악 끝") // 루프가 종료된 후 이 메시지가 출력됩니다.
````

- **코드 설명**:
    - `var counter = 0`: `counter`라는 변수를 만들고 0으로 초기화합니다. 이 변수는 루프가 몇 번 실행되었는지 세는 역할을 합니다.
    - `while true { ... }`: 이것이 무한 루프입니다. 이 중괄호 `{}` 안의 코드는 `break`를 만나기 전까지 계속 반복 실행됩니다.
    - `print(" ")`: 루프가 한 번 반복될 때마다 공백 문자 하나를 출력합니다. 이것이 '4분 33초'의 한 순간을 나타낸다고 가정합니다.
    - `counter += 1`: `counter` 변수의 값을 1 증가시킵니다. 루프가 실행된 횟수를 기록합니다.
    - `if counter == 273 { ... }`: `counter`의 값이 273과 같은지 확인하는 조건문입니다. (273번은 예시를 위해 정한 임의의 횟수입니다.)
    - `break`: `if` 조건(`counter == 273`)이 참이 되면, 이 `break` 키워드가 실행됩니다. `break`는 현재 실행 중인 `while` 루프를 즉시 중단하고 루프 바로 다음 코드로 프로그램 실행 흐름을 이동시킵니다.
    - `print("음악 끝")`: `while` 루프가 `break`에 의해 종료된 후에 실행되는 코드입니다.
- **실행 결과**: 이 코드를 실행하면 공백 문자가 273번 출력된 후, 마지막에 "음악 끝" 이라는 메시지가 출력됩니다.

---

## 🔑 핵심 정리 (Key Summary)

- **`while true`**: 조건을 항상 참으로 만들어 무한 루프를 생성합니다.
- **`break`**: 루프 실행 중에 특정 조건이 만족되었을 때, 해당 루프 전체를 즉시 종료하고 루프 다음 코드로 넘어갑니다.
- 무한 루프를 사용할 때는 프로그램이 멈추지 않도록 반드시 루프 내부에 `break` 등을 이용한 **종료 조건을 포함해야 합니다.**


# Swift - 루프 항목 건너뛰기 (`continue`) (Skipping Loop Items)

## 🎯 개요 (Overview)

* 이전에 보셨듯이, `break` 키워드는 루프 전체를 종료하고 빠져나옵니다.
* 하지만 루프를 완전히 종료하는 대신, 현재 처리 중인 항목(또는 현재 반복)만 건너뛰고 다음 항목으로 계속 진행하고 싶을 때가 있습니다.
* 이럴 때 바로 `continue` 키워드를 사용합니다.

---

## 🛠️ 사용 방법 (How to Use)

* `continue`는 주로 루프 안의 특정 조건문 (`if` 등) 내에서 사용됩니다.
* 루프가 실행될 때, 어떤 조건이 참이 되어 `continue` 문을 만나면, Swift는 현재 반복에서 `continue` 아래에 있는 코드들을 모두 무시하고 즉시 루프의 다음 반복으로 이동합니다.

---

## 🔢 예시: 홀수 건너뛰고 짝수만 출력하기 (Example: Skipping Odd Numbers and Printing Only Even Numbers)

`continue`가 어떻게 작동하는지 알아보기 위해, 1부터 10까지 반복하는 루프를 작성하고, Swift의 나머지 연산자를 사용하여 홀수인 숫자는 건너뛰고 짝수만 출력하는 예시를 만들어 보겠습니다.

### 예시 코드 및 설명

```swift
// 1부터 10까지 반복하는 for-in 루프 시작
for i in 1...10 { // i는 1부터 10까지 순서대로 숫자를 가집니다.
    
    // 현재 숫자가 홀수인지 확인하는 조건문
    // Swift의 나머지 연산자 (%)를 사용합니다.
    // 나머지 연산자는 해당 숫자에 2가 몇 번 들어가는지 계산한 후,
    // '남는 값'을 반환합니다.
    if i % 2 == 1 { // 숫자를 2로 나눈 나머지가 1과 같다면, 즉 숫자가 홀수이면...
        
        // 홀수이므로 현재 반복을 건너뜁니다.
        continue // 이 키워드를 만나면 아래 print(i) 코드를 실행하지 않고 다음 반복(i가 다음 숫자로 넘어감)으로 즉시 이동합니다.
    }
    
    // 만약 위 if 조건이 거짓이었다면 (즉, 숫자가 짝수라면)...
    // 이 코드가 실행되어 현재 숫자를 출력합니다.
    print(i) // continue에 의해 건너뛰어지지 않은 숫자 (즉, 짝수)만 출력됩니다.
}

// 루프가 10까지 모두 실행된 후, 루프 다음 코드가 실행됩니다.
print("루프 끝")
````

- **코드 설명**:
    - `for i in 1...10 { ... }`: 1부터 10까지의 숫자를 순서대로 `i`라는 변수에 할당하며 루프를 10번 반복합니다.
    - `if i % 2 == 1 { ... }`: 현재 숫자 `i`가 홀수인지 확인하는 조건문입니다. **나머지 연산자 (`%`)**를 사용하여 `i`를 2로 나눈 나머지를 구하고, 그 결과가 1과 같은지 비교합니다. 나머지가 1이면 숫자는 홀수입니다.
    - `continue`: `if` 조건(숫자가 홀수)이 참일 경우 이 `continue` 키워드가 실행됩니다. `continue`가 실행되면 현재 `i` 값에 대해 `print(i)` 코드는 건너뛰어지고, 루프는 즉시 다음 숫자(다음 반복)로 넘어갑니다.
    - `print(i)`: 이 코드는 `if i % 2 == 1` 조건이 거짓일 경우에만 실행됩니다. 즉, `i`가 2로 나누어 떨어져서 나머지가 1이 아닌 경우 (나머지가 0인 경우, 즉 짝수인 경우)에만 이 코드가 실행되어 `i`의 값이 출력됩니다.
- **실행 결과**: 이 코드를 실행하면 `continue`에 의해 홀수인 1, 3, 5, 7, 9는 건너뛰어지고, 짝수인 2, 4, 6, 8, 10만 출력될 것입니다. 마지막에 "루프 끝" 이라는 메시지가 출력됩니다.

---

## 🔑 핵심 정리 (Key Summary)

- **`break`**: 루프를 완전히 중단하고 루프 다음 코드로 이동합니다.
- **`continue`**: 루프의 현재 반복만 건너뛰고 다음 반복으로 이동합니다.
- `continue`는 루프 내에서 특정 조건에 해당하는 항목만 제외하고 나머지 항목들을 처리하고 싶을 때 유용하게 사용됩니다.
