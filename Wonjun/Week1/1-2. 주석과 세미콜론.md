# Description
## 1. 주석
: 코드에서 설명 또는 기록을 위해 실행되지 않는 문자를 추가할때 사용, 컴파일러가 무시한다.
```swift
//This is comment.
```
### 주석의 종류
- 한 줄 주석 `//`
- 여러 줄 주석  `/**/`
### 1-1. 주석을 활용한 Notation
- Notation을 통해서 주석별로 검색 가능!

```swift
주석 종류
// MARK: - 내용
// TODO: - 내용
// FIXME: - 내용
```

-----------
## 2. 세미콜론
: 여러 구문을 한줄로 작성할 경우 세미콜론은 필수로 작성되어야 한다.
```swift
let cat = "meow"; print(cat)
// Prints "meow"
```
