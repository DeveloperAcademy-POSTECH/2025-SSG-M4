## Description

# 정수
- Swift는 8,16,31,64bit 형태의 signed & unsigned 정수를 지원한다.

# 정수 범위 
- `min`, `max` 프로퍼티를 통해 각 정수 타입의 최소값과 최대값을 가져올 수 있다.

# Int
- 현재 플랫폼의 네이티브 사이즈와 같은 `Int` 정수 타입을 제공한다.

# UInt
- - 현재 플랫폼의 네이티브 사이즈와 같은 `UInt` 정수 타입을 제공한다.
